# CS111A Section 5
_CS111A Code Repository for Section 5 (Winter 2023)_

Welcome back to CS111A! Today we'll be talking about Multiprocessing + Pipes!... 

To clone this repository, navigate to where you want the project to live, and then type

`git clone https://github.com/tmaster628/WIN23_CS111ACE_S5.git`

From there, `cd` into the repo (and either the `simple_print` or `autograder` folders) and type `make`. Now you should be able to run the available programs!
